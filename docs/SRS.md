## Project

Mood-Based Playlist Generator

## Summary

The application recommends music playlists based on a user's mood. Users will later be able to select a mood manually or enter journal-style text. The app will generate playlists using a music API such as Spotify or YouTube.

## Planned Capabilities

- Manual mood selection
- Journal-style text input for mood detection
- Dynamic mood themes
- Playlist results
- Authentication
- Favorite moods
- Playlist history
- Sharing
- Dashboard
- Sentiment analysis

## Phase 1 Scope

Phase 1 creates the full-stack project foundation only:

- React TypeScript Vite frontend
- Express TypeScript backend
- Prisma PostgreSQL schema foundation
- Placeholder frontend pages and components
- Placeholder backend module structure
- Health endpoint
- Static moods endpoint
- Documentation

## Out Of Scope For Phase 1

- Final UI
- Real Spotify or YouTube API integration
- Authentication implementation
- JWT flow
- Favorite moods behavior
- Playlist history behavior
- Journal sentiment analysis
- Dashboard analytics
- Deployment
